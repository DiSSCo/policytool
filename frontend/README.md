# Policy Tool Frontend

This folder contains the source for the frontend app of the policy tool.
It is written in Vue 3 using Vite as the build tool.
CSS is handled with Tailwind.

## Install
```sh
npm install
```

## Run dev server
Runs a development server using Vite allowing for rapid development work.
Should autolaunch in the browser, but if not, it's accessible at http://localhost:3000.

```sh
npm run dev
```

## Build
This will build the production artefacts and place them in the django static folder.

```sh
npm run build
```
